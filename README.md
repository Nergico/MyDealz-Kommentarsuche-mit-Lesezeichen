# MyDealz-Kommentarsuche-mit-Lesezeichen
Volltextsuche für MyDealz-Kommentare – ganz ohne Browser-Erweiterung! Dieses Lesezeichen (Bookmarklet) durchsucht selbst hunderte Antworten in Millisekunden dank lokalem Cache. Bietet praktische Filter, Dark-Mode und Highlighting. Einfach im Browser als Lesezeichen speichern und die Suchleiste auf jeder Deal-Seite per Klick öffnen.
