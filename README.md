**MyDealz Kommentar-Volltextsuche 🔖 (Lesezeichen / Bookmarklet Edition)**  
Dieses Skript rüstet eine extrem praktische Funktion für MyDealz nach: Eine echte Volltextsuche für Kommentare innerhalb eines Deals oder einer Diskussion.  

Wer MyDealz kennt, weiß: Bei beliebten Deals mit hunderten Antworten dauert es oft ewig, bestimmte Informationen zu finden. Dieses Skript löst das Problem, indem es alle Kommentare ausliest, in einem lokalen Cache zwischenspeichert und blitzschnell durchsuchbar macht.  
Das Besondere an diesem Repository: Du benötigst absolut keine Browser-Erweiterung (wie Tampermonkey). Das Skript läuft als einfaches Lesezeichen (Bookmarklet) komplett nativ in deinem Browser!  
Ich habe auch Versionen mit Tampermonkey erstellt, was ich persönlich angenehmer finde: https://github.com/Nergico/MyDealz-Kommentarsuche-Tampermonkey

**🚀 Die 3 Varianten (Wähle deinen Favoriten)**  
Je nachdem, wie du MyDealz am liebsten nutzt, stehen drei verschiedene Versionen des Lesezeichens zur Verfügung. Wähle einfach den Code-Block deiner gewünschten Variante aus.  

**🌟 Variante 1: Pro-Suche (Die aktuellste Version mit erweiterten Funktionen)**  
Die modernste und umfangreichste Version. Sie integriert sich nahtlos in die MyDealz-Seite und bietet dir zusätzlich schnelle Klick-Filter (z. B. "Letzte 24h", "1+ Reactions") sowie eine extrem übersichtliche "Mini-Ansicht" für gefundene Kommentare. Über einen kleinen Button unten rechts kannst du jederzeit sofort zur Suchleiste zurückspringen.

**🔹 Variante 2: Inline-Suche (Klassisch in der Seite)**  
Diese Version ist etwas schlanker und fügt eine elegante Suchleiste direkt über dem obersten Kommentar ein (genau dort, wo du auch die Kommentare über "sortiert nach" sortieren kannst). Alles bleibt auf einer Seite. Dafür werden nur die Kommentare mit den gefilterten Wörtern wiedergegeben und keine Kommentare die dazugehören.

**🔹 Variante 3: Sofort-Suche (Im neuen Tab)**  
Diese Version ist fast wie Vesrion 3, nur dass hier direkt ein neuer Tab geöffnet wird und dort alles weitere passiert.  


**🛠️ So richtest du das Lesezeichen ein**  
Die Installation dauert nur wenige Sekunden und funktioniert in allen gängigen Browsern (Chrome, Edge, Safari, Firefox).  
Mache in deinem Browser einen Rechtsklick auf deine Lesezeichenleiste und wähle Lesezeichen hinzufügen (oder "Seite hinzufügen").  
Gib dem Lesezeichen einen aussagekräftigen Namen (z. B. MyDealz Suche).  
Kopiere den kompletten, langen Code-Block aus deiner gewählten Variante (siehe oben).  
Füge den gesamten Code in das Feld URL (oder "Adresse") des neuen Lesezeichens ein.  
Klicke auf Speichern.  
Anwendung: Öffne einfach einen beliebigen Deal oder eine Diskussion auf MyDealz und klicke auf dein neues Lesezeichen!  

**✨ Weitere Features & Wichtige Hinweise**  
Lokaler Cache (Rasante Geschwindigkeit): Wenn du in einem Deal suchst, werden die Kommentare einmalig heruntergeladen. Jede weitere Suche im selben Deal passiert danach in Millisekunden. Dafür dauert es Anfangs etwas, bis alle Kommentare geladen werden, vor allem bei vielen Kommentaren.  
Firefox-Kompatibilität: Die bekannte "Firefox-Macke" (die bei älteren Bookmarklets oft zu einer weißen Fehler-Seite führte) wurde hier technisch berücksichtigt. Die Lesezeichen laufen auch im Firefox einwandfrei.  
Dark Mode Support: Die Skripte passen sich automatisch an das MyDealz-Theme (Hell/Dunkel) an, damit du auch nachts blendfrei suchen kannst.  
Wichtig zu Variante 3 (Pop-ups): Wenn du Variante 3 nutzt und kein Suchfenster erscheint, prüfe bitte deine Browsereinstellungen. Das Skript benötigt die Berechtigung, für www.mydealz.de Pop-up-Fenster öffnen zu dürfen.
